## Overview

In Zone 3, you'll design and orchestrate an **agent system** using **Azure AI Foundry**.

Real-world agent systems are rarely a single agent solving a single problem.

They are systems of responsibility ‚Äî agents, workflows, and data working together with clear boundaries.

This zone shifts your focus from building agents to designing agent systems.

It brings together everything you built in earlier zones:

- **Zone 1:** Data grounding and retrieval  
- **Zone 2:** Reasoning, routing, and conversation flow  

Rather than focusing on a single agent or scripted scenario, this zone focuses on **system-level design** ‚Äî how agents, workflows, and data work together to solve a real problem.

## What success looks like

By the end of this zone, you'll have:

- A clearly defined **real-world scenario**
- At least one **grounded agent** using the provided data
- A **workflow** that orchestrates agent behavior
- A design that could realistically extend to real data or systems

You aren't expected to fully build or automate everything.
